# installing Mongo Exporter on K8s Using helm chart

1, make sure helm is install on the node or k8s jump server

2, add the helm repo to local helm repo list

    # helm repo add prometheus-community https://prometheus-community.github.io/helm-charts

3, install the chart using the follwing command

    # helm install elst-mongo-exporter prometheus-community/prometheus-mongodb-exporter
